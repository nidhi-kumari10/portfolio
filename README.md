# 🌐 Personal Portfolio – Kumari Nidhi

A modern and responsive **Full Stack Developer Portfolio** built using React and Tailwind CSS. This portfolio showcases my projects, technical skills, and provides a fully functional contact section for easy communication.

---

## 🚀 Live Demo

👉 https://portfolio-lemon-mu-25.vercel.app/

---

## ✨ Features

* 👩‍💻 Interactive and modern UI design
* 📱 Fully responsive across all devices
* ⚡ Smooth scrolling navigation
* 🎯 Projects showcase with live links & GitHub repos
* 🛠️ Categorized technical skills section
* 📩 Functional contact form using EmailJS
* 🔗 Social links (GitHub, LinkedIn, Email)
* 🎨 Consistent theme with animations and hover effects

---

## 🛠️ Tech Stack

* ⚛️ React.js
* ⚡ Vite
* 🎨 Tailwind CSS
* 📩 EmailJS (for contact form)
* 🌐 Vercel (deployment)

## ⚙️ Installation & Setup

Clone the repository:

```
git clone https://github.com/nidhi-kumari10/portfolio.git
```

Navigate to project folder:

```
cd portfolio
```

Install dependencies:

```
npm install
```

Run the project:

```
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file in the root and add:

```
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
```
